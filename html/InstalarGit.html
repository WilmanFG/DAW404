¿Como instalar un Cliente Git en Windows?



Git es un sistema de control de versiones usado en el desarrollo de software para guardar los cambios realizados en un proyecto, mantener un historial y navegar a través del mismo en cualquier momento.

1. Ingresa a la página oficial de Git

Hacer click al botón de Downloads for Windows.

2. Ejecutar Instalador

Una vez descargada, dale click al archivo con el formato Git-version.exe. Al inicio preguntará por permisos para ejecutar el instalador, a lo cual debemos responder que sí.

3. Establecemos la ruta donde será instalado

Como muchos de los instaladores en Windows, debemos de aceptar las opciones por defecto y darle Next (siguiente) a todo hasta que nos salga el botón de instalar.




4. Componentes de Git

Aquí podemos seleccionar los componentes de Git que ofrece para ser instalados

5. Nombre de la carpeta

El instalador de Git, nos permite cambiar el nombre con el que luego lo buscaremos en la lista de programas. Se recomienda dejar el nombre por defecto

6. Terminales

En este paso podemos seleccionar la terminal que nos convenga para ejecutar Git. Se recomienda dejar la opción por defecto

7. Seleccionar el formato para lineas finales

El instalador de Git nos da tres opciones para configurar el formato para los finales de línea de los archivos




8. Emulador de Git Bash

Seleccionamos el tipo de emualador que usara Git Bash

9. Opciones extra de Git

Encontraremos opciones extra como el almacenamiento de cache de los archivos, enlaces simbólicos y el administrador de credenciales de Git

10. Nuevas herramientas de Git

Se recomienda dejar como viene por defecto




11. Instalando

Una vez terminada la configuración del instalador de Git, comenzará a instalar todos los archivos necesarios en la carpeta indicada al inicio.

12. Ejecutar Instalador

Al finalizar la instalación, nos dará la opción de abrir Git Bash para finalizar.

13. Instalación completada y lista para usar

Al finalizar la instalación, nos dará la opción de abrir Git Bash para finalizar. Y ésta será la terminal que usaremos para ingresar comandos Unix y de Git
